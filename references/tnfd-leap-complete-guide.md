## Step 1: TNFD 是什么？（1 分钟了解）

### 核心定义

**TNFD**（Taskforce on Nature-related Financial Disclosures，自然相关财务披露工作组）是一个全球性倡议，旨在帮助企业**识别、评估、管理和披露**与自然相关的风险和机遇。

### 方法论基础

TNFD 框架构建于以下国际标准和框架之上：

| 框架/标准 | 发布机构 | TNFD 整合方式 |
|----------|---------|-------------|
| **自然资本议定书** | Capitals Coalition | LEAP 框架的方法论基础 |
| **TCFD 框架** | TCFD | 四支柱披露结构（治理/战略/风险/指标） |
| **ISSB 准则** | IFRS Foundation | 与 S1/S2 准则对齐 |
| **SBTN** | Science Based Targets Network | 目标设定方法 |
| **ENCORE** | UNEP-WCMC/Global Canopy | 行业依赖/影响矩阵 |

### 自然资本议定书（Natural Capital Protocol）

**TNFD 的评估方法论直接来源于自然资本议定书**。

| 项目 | 详情 |
|------|------|
| **发布机构** | Capitals Coalition（多元资本联盟） |
| **发布时间** | 2016 年 7 月 |
| **框架结构** | 4 阶段 9 步骤（WHY-WHAT-HOW-WHAT NEXT） |
| **与 TNFD 关系** | TNFD LEAP 框架构建于此之上 |
| **官方下载** | [PDF 下载](https://capitalscoalition.org/capitals-approach/natural-capital-protocol/) |

**自然资本议定书 4 阶段 9 步骤**：
```
第一阶段：WHY（为什么）
├─ 步骤 1：理解自然资本与企业的关系
└─ 步骤 2：确定评估目的

第二阶段：WHAT（是什么）← 对应 TNFD Locate + Evaluate
├─ 步骤 3：确定业务范围
├─ 步骤 4：确定自然资本影响和依赖
└─ 步骤 5：选择评估方法

第三阶段：HOW（怎么做）← 对应 TNFD Evaluate + Assess
├─ 步骤 6：收集数据
└─ 步骤 7：分析影响和依赖

第四阶段：WHAT NEXT（下一步）← 对应 TNFD Assess + Prepare
├─ 步骤 8：评估风险和机遇
└─ 步骤 9：制定行动计划
```

**牧原股份案例**：牧原在 2025 年度 ESG 报告中明确引用《自然资本议定书》实质性分析标准，结合 ENCORE 数据库评估业务活动对自然的影响和依赖程度。

**详细解读**：详见 `references/natural-capital-protocol-guide.md`

### 为什么重要？

| 驱动因素 | 说明 |
|---------|------|
| **监管要求** | 欧盟 CSRD、英国强制披露、中国《上市公司可持续发展报告指引》 |
| **投资者需求** | 贝莱德、先锋等机构投资者要求自然风险评估 |
| **供应链压力** | 苹果、沃尔玛等要求供应商披露自然影响 |
| **风险管理** | 生物多样性丧失是未来 10 年最大全球风险之一（WEF 2024） |

### 与 TCFD 的关系

```
┌─────────────────────────────────────────────────────┐
│              TCFD vs TNFD 对比                       │
├─────────────────────────────────────────────────────┤
│                                                     │
│  TCFD（气候相关）          TNFD（自然相关）          │
│  ├─ 温室气体排放          ├─ 生物多样性影响         │
│  ├─ 气候风险              ├─ 生态系统依赖           │
│  ├─ 转型风险              ├─ 物理风险 + 转型风险    │
│  └─ 四支柱框架            └─ 四支柱框架（相同）     │
│                                                     │
│  ✅ TNFD 与 TCFD 高度兼容，已完成 TCFD 的企业可快速上手  │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 核心框架：LEAP 四阶段

| 阶段 | 核心问题 | 输出物 | 预计时间 |
|------|---------|--------|---------|
| **L**ocate（定位） | 我们的资产/供应链在哪里？ | 资产坐标清单 + 空间风险地图 | 1-2 周 |
| **E**valuate（评价） | 我们对自然的依赖和影响是什么？ | 行业依赖矩阵 + 影响驱动因素 | 1-2 周 |
| **A**ssess（评估） | 这些带来什么财务风险/机遇？ | 风险量化分析 + 机遇清单 | 2-4 周 |
| **P**repare（准备） | 如何披露？如何响应？ | TNFD 报告 + 响应策略 | 2-4 周 |

**总周期**：首次披露约 8-12 周（取决于数据可用性）

## Step 2: 谁需要披露？

### 强制披露（中国）

根据《上市公司可持续发展报告指引》（2024 年 4 月）：

**强制主体**（最晚 2026 年 4 月 30 日前披露）：
- ✅ 上证 180 指数成分股
- ✅ 科创 50 指数成分股
- ✅ 深证 100 指数成分股
- ✅ 创业板指数成分股
- ✅ 境内外同时上市公司

**约 400 家企业**属于强制披露范围。

### 自愿披露

| 类型 | 建议披露 |
|------|---------|
| 供应链企业 | 苹果/沃尔玛等要求供应商披露 |
| 金融机构 | 投资者要求 + 监管预期 |
| 高影响行业 | 农业、采矿、能源、林业 |
| 出口企业 | 欧盟 CSRD 供应链要求 |

### 自我评估问题

```
1. 我们是否是上述强制披露主体？
2. 我们的客户/投资者是否要求 TNFD 披露？
3. 我们的业务是否高度依赖自然（如农业、林业）？
4. 我们的业务是否对自然有重大影响（如采矿、能源）？

→ 如果任一答案为「是」，建议启动 TNFD 披露准备
```

## Step 3: LEAP 四阶段详解

### 阶段 L：Locate（定位）

**核心任务**：识别企业与自然的接触点

**具体步骤**：
1. 列出所有运营地点（工厂、办公室、仓库等）
2. 收集每个地点的经纬度坐标
3. 识别供应链关键节点（供应商位置）
4. 在地图上叠加生态敏感区（保护区、生态红线等）

**输出物**：
- 资产坐标清单（CSV/Excel）
- 空间风险地图（可可视化）
- 高风险点位清单（邻近保护区/生态红线的资产）

**工具推荐**：
- WDPA Protected Planet（保护区查询）
- 地理遥感生态网（中国区域数据）
- Google Earth / 高德地图（坐标获取）

**预计时间**：1-2 周

---

### 阶段 E：Evaluate（评价）

**核心任务**：评估企业对自然的依赖和影响

**具体步骤**：
1. 确定行业分类（ISIC 代码）
2. 查询 ENCORE 数据库的行业依赖/影响矩阵
3. 识别高依赖度的生态系统服务（如水供应、土壤质量）
4. 识别高影响驱动因素（如土地利用变化、水排放）
5. 进行重要性排序

**输出物**：
- 行业依赖矩阵（来自 ENCORE）
- 重要性议题清单
- 优先关注的依赖和影响

**工具推荐**：
- ENCORE 数据库（https://encorenature.org）
- SBTN 重要性筛选工具
- CASS-ESG 6.0 指标库

**预计时间**：1-2 周

---

### 阶段 A：Assess（评估）

**核心任务**：将依赖和影响转化为财务风险和机遇

**具体步骤**：
1. 识别物理风险（如水资源短缺影响生产）
2. 识别转型风险（如新法规增加合规成本）
3. 识别机遇（如生态修复带来品牌价值）
4. 量化财务影响（收入影响、成本增加、资本支出）
5. 评估风险发生概率和时间范围

**输出物**：
- 风险登记册（含概率、影响、时间）
- 机遇清单
- 财务影响量化分析

**工具推荐**：
- EY VaR for Nature（风险量化）
- NGFS 自然风险框架
- 情景分析工具

**预计时间**：2-4 周

---

### 阶段 P：Prepare（准备）

**核心任务**：整合到决策流程并准备披露

**具体步骤**：
1. 制定响应策略（风险缓解、机遇捕捉）
2. 设定目标和 KPI（参考 SBTN）
3. 整合到企业风险管理流程
4. 编写 TNFD 报告（对标 14 项披露建议）
5. 准备第三方鉴证（如需要）

**输出物**：
- TNFD 披露报告
- 响应策略和行动计划
- 目标和 KPI 框架

**工具推荐**：
- TNFD 报告模板
- CASS-ESG 6.0 报告指南
- 四大鉴证方法论

**预计时间**：2-4 周

## Step 5: 注册指南

### WDPA Protected Planet 注册（5 分钟）

**网址**：https://www.protectedplanet.net

**步骤**：
1. 点击右上角「Sign In」
2. 选择「Create an account」
3. 填写邮箱、密码、姓名
4. 验证邮箱
5. 登录后进入「Download」页面
6. 选择「WDPA Shapefile」或「GeoJSON」
7. 同意使用条款 → 开始下载

**数据格式**：Shapefile / GeoJSON  
**文件大小**：约 600MB（全球）  
**更新频率**：每月

---

### WRI Aqueduct 注册（3 分钟）

**网址**：https://www.wri.org/aqueduct

**步骤**：
1. 点击「Get the Data」
2. 填写姓名、邮箱、机构
3. 选择数据用途（学术/商业/政府）
4. 提交后自动获得下载链接
5. 选择水风险指标下载

**数据格式**：CSV / GeoTIFF  
**文件大小**：约 50-200MB  
**更新频率**：年度

---

### ENCORE 数据下载（无需注册）

**网址**：https://encorenature.org/en/data-and-methodology/methodology

**步骤**：
1. 滚动到页面底部「Download the updated ENCORE knowledge base」
2. 点击「DOWNLOAD」按钮
3. 自动下载 ZIP 文件（约 2MB）
4. 解压后获取 CSV 文件

**数据格式**：CSV / Excel  
**文件大小**：约 2MB  
**更新频率**：年度

---

### 国家生态数据中心（中国）

**网址**：http://nesdc.org.cn

**步骤**：
1. 浏览数据目录
2. 选择需要的数据集
3. 部分数据直接下载，部分需申请
4. 申请需填写研究用途说明

**数据格式**：多种（CSV、Shapefile、NetCDF）  
**文件大小**：因数据集而异  
**更新频率**：不定期

## Step 7: 披露模板

### TNFD 14 项披露建议（四支柱）

**治理（2 项）**：
1. 董事会如何监督自然相关风险和机遇？
2. 管理层如何评估和管理自然相关风险和机遇？

**战略（2 项）**：
3. 自然相关问题对业务模式、价值链、战略和财务规划的影响
4. 自然相关机遇对业务模式、价值链、战略和财务规划的影响

**风险与影响管理（4 项）**：
5. 识别自然相关风险的流程
6. 评估自然相关风险的方法
7. 如何将自然风险管理整合到整体风险管理
8. 管理自然影响的流程

**指标与目标（6 项）**：
9. 用于评估和管理自然相关风险和机遇的指标
10. 自然相关目标及其进展
11. 指标计算方法的变更
12. 自然相关财务影响量化
13. 资本配置与自然风险的对齐
14. 内部自然定价（如适用）

## 常见问题（FAQ）

### Q1: 我们没有坐标数据，怎么办？

**A**: 可以用以下方式获取：
- Google 地图：右键点击位置 → 复制坐标
- 高德地图：搜索地址 → 查看经纬度
- 批量转换：使用在线工具将地址批量转为坐标

### Q2: ENCORE 数据是英文的，有中文版吗？

**A**: ENCORE 官方暂无中文版，但：
- TNFD Skill 可提供中文解读
- CASS-ESG 6.0 有中文行业对照
- 可使用翻译工具辅助理解

### Q3: 我们需要第三方鉴证吗？

**A**: 
- 中国《指引》暂未强制要求鉴证
- 但建议进行内部数据质量检查
- 四大提供 TNFD 鉴证服务（安永、普华永道等）

### Q4: TNFD 和 CASS-ESG 6.0 是什么关系？

**A**: 
- TNFD 是国际框架
- CASS-ESG 6.0 是中国本土实操指南
- CASS-ESG 6.0 已整合 TNFD 要求
- 建议同时参考两者

### Q5: 多久需要更新一次披露？

**A**: 
- 建议年度更新（与 ESG 报告同步）
- 重大变化（如新工厂、并购）应及时更新
- LEAP 评估建议每 2-3 年全面重做
